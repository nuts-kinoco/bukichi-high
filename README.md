# 🦑 ブキチハイ！ (bukichi-high!)

スプラトゥーン3対応！超高機能ブキ＆ルール＆ステージランダム抽選ツール「ブキチハイ！」です。🚀

ライト/ダークテーマの滑らかな切り替え、プレイヤーの動的な追加・削除、ブキの未所持問題を解決する「バッファ（予備ブキ）表示」、スリル満点の「ロシアンルーレット（他薦）モード」、独自の「特殊ルール（縛り）抽選」、そしてワンクリックでDiscord等のチャットに貼り付けられる「プレーンテキスト出力」を搭載した、極めて洗練されたシングルページアプリケーション（SPA）です。

---

## ✨ 主な機能

1.  **🎮 プレイヤーの動的追加・削除**
    *   初期表示は10名。＋ボタンで何人でも追加でき、❌ボタンで削除も可能。名前を入力したプレイヤーのみを対象に抽選します。
2.  **🎁 バッファ（予備ブキ）表示機能（デフォルト0）**
    *   ブキを全種類持っていないプレイヤーがいても安心！設定した個数分の予備ブキを別枠で同時に重複なし抽選し、交換用の候補として画面に並べます。
3.  **💥 ロシアンルーレット（他薦）モード**
    *   「ブキロシアンルーレット」「ルールロシアンルーレット」を個別にON/OFF可能。
    *   ONにすると、指定した確率（10%〜100%）でブキやルールが **「[他の誰か] さんが指定」** に置き換わり、プラベでのコミュニケーションが圧倒的に盛り上がります！他薦が発生したカードは警告オレンジネオンで点滅します。
4.  **⚙️ 特殊ルール（縛り）抽選＆ブラウザ永続管理**
    *   「サブウェポン使用禁止」や「ジャイロOFF縛り」などの特殊対戦ルールをランダムで抽選します。
    *   初期のCSVデータロードに加え、UI上からユーザー自身が新しい特殊ルールを自由に追加・削除可能。追加したカスタムデータはブラウザ（`localStorage`）に永続保存され、リロードしても消えません。
5.  **☀️/🌙 コトバトラー風ライト・ダークテーマ切り替え**
    *   ヘッダーの太陽と月のトグルボタンで、画面テーマを滑らかに切り替えます。選択状態はブラウザに記憶されます。
6.  **📋 Discord等への簡単コピペ機能**
    *   抽選結果を絵文字なしのシンプルな改行テキストとして出力。ワンクリックでクリップボードにコピーできます。コピー完了時はネオンカラーのトースト通知が作動します。
7.  **📊 CSVによる簡単なブキ・ルール管理**
    *   `weapons.csv` や `special_rules.csv` を編集するだけで、プログラムを一切触らずにブキや初期の特殊ルールを追加・削除・編集できます。
8.  **📱 スマートフォン完全対応**
    *   レスポンシブデザインにより、スマートフォンやタブレットの狭い画面でも一切レイアウトが崩れず、極めて快適に操作・コピペが可能です。

## 🚀 使い方とGitHub Pagesへのデプロイ手順

このアプリは静的ファイルのみで構成されているため、サーバー側のプログラムは不要です。ブラウザだけで完全に動作します。

### 1. ローカルで起動する
1. `bukichi-high!` ディレクトリをアクティブなワークスペースとしてエディタ等で開きます。
2. `index.html` をブラウザで直接開くか、ローカルサーバーを立ち上げてアクセスします。
   *(※CSVファイルをJavaScriptの `fetch` を使用して非同期で読み込んでいるため、ブラウザによっては直接ファイルを開くとセキュリティ制限にかかる場合があります。その際はローカルWebサーバーのご利用を推奨します。)*

### 2. 🎉 GitHub Pagesにデプロイして一般公開する
このアプリをネット上に公開し、いつでもスマホやPCから友達とアクセスできるようにする手順です。

1.  **リポジトリを作成**
    *   GitHubアカウントにアクセスし、新しいパブリック（Public）リポジトリを作成します。
    *   リポジトリ名は **`bukichi-high`** がお勧めです。
2.  **ファイルをプッシュ**
    *   本フォルダ内のすべてのファイルをそのリポジトリにコミット＆プッシュします。
3.  **GitHub Pagesの設定をオンにする**
    *   リポジトリの右上にある **「Settings（設定）」** をクリックします。
    *   左側のメニューから **「Pages」** を選択します。
    *   **「Build and deployment」** の中にある **「Branch」** を `None` から `main` (または `master`) に変更し、右側の **「Save」** ボタンを押します。
4.  **公開完了！**
    *   約1〜2分待つと、画面上部に公開URL（例: `https://<あなたのユーザー名>.github.io/bukichi-high/`）が表示されます。そのURLをブックマークしたり、対戦メンバーに送るだけで、全員で同じツールを使用できます。

## 🛠️ カスタマイズ（データの編集）

*   **ブキデータの追加**:
    `weapons.csv` の最下行に、以下のフォーマットで1行追加して保存するだけで自動反映されます。
    ```csv
    カテゴリー,ブキ名
    シューター,新ブキネオ
    ```
*   **初期特殊ルールの追加**:
    `special_rules.csv` の最下行に、新しいルール名を1行追加して保存するだけで自動反映されます。
    ```csv
    ルール名
    新縛りルール
    ```
